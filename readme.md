# Comandos de GitFlow

* git branch -> visualizar as braches
* git branch nome -> criando branch
* git switch nome -> entrando na branch
* git switch - c nome -> criando e entrando na branch
* git checkout -b nome -> criand e entrando na branch
* git merge none -> faz a junção das branches
